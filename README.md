# my dev blog

a blog about web development, programming, and technology.

## running locally

```bash
pnpm install
pnpm dev # Runs dev server at http://localhost:3000
pnpm db:start # Runs Supabase at http://localhost:54323/
```

create `.env.local` file and add values for following keys:

```
SUPABASE_URL=
SUPABASE_SECRET_KEY=
```
